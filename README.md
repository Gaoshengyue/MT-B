## MeowTheory(喵态)后台

### 项目启动
#### 包安装
``pip3 install -r requirements.txt``
#### 数据化迁移
```python3 manager.py makemigrations```
```python3 manager.py migrate```

#### settings配置
```/MT/settings.py```
#### 默认头像存储位置
```/media```

#### 应用视图
```/MeowTheory/views.py```
